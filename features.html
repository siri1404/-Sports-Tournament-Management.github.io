Features of the Application

This application will be capable of scheduling the timings for various sporting events for different categories and making sure that the events for different categories do not clash.

This application also ensures that there is fair evaluation of points and the score tables remain upto date after each of the events. It makes sure that there is no calculation errors that arises during the tallying of scores.

It also helps us know which teams are at the top of the table at all times. This task would be cumbersome for human calculation as certain sporting events such as team events have more weightage than individual sporting events.
