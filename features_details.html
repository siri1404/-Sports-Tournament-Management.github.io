Abstract

he first feature of the application is slotting of matches. The application checks if the time slot entered for any particular sport of any particular category is valid or not.

There are four different categories considered:


• UG Boys
• UG Girls
• PG Boys
• PG Girls

Two different matches of the same category cannot be conducted simultaneously. Two matches cannot be conducted in the same court/pitch simultaneously.

The application ensures that the points are added without any error or delay. Any changes in the points or results in the tables for the individual sports will be reflected accordingly in the main points_table immediately.

For the addition of points we have to keep in mind the following criteria:

• Winning group games, ie basket ball and volley ball, gives a team double points.
• Winning individual games, ie badminton and table tennis, gives a team single points.
• If a particular college fails to put forward a team for any particular sport, they will receive negative points.
• In case there is a tie in the scores of any two teams, the team with higher number of wins gets a higher position in the table.

At the end of the tournament, we run a query to find the top 3 teams and then declare the prizes.
